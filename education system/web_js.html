1. Introduction to Java

What is Java?

Definition: Java is a high-level, object-oriented programming language designed to be platform-independent. It is widely used for building enterprise-scale applications, web applications, and Android apps.

Key Features:

  • Object-Oriented: Encourages modular programming with classes and objects.
  • Platform-Independent: Java programs run on any device with a Java Virtual Machine (JVM).
  • Strongly Typed: Requires explicit declarations of data types.
  • Automatic Memory Management: Uses garbage collection to manage memory.

Basic Java Syntax

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

2. Introduction to DOM Manipulation

What is the DOM?

Definition: The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing scripts to dynamically access and update the content, structure, and style of documents.

JavaScript and DOM Manipulation

Selecting Elements:

var element = document.getElementById('elementId');
var elements = document.getElementsByClassName('className');
var queryElement = document.querySelector('.className');

Modifying Content:

element.innerHTML = 'New Content';
element.textContent = 'New Text Content';

Changing Attributes:

element.setAttribute('attributeName', 'value');

Styling Elements:

element.style.color = 'blue';
element.style.backgroundColor = 'yellow';

Creating and Appending Elements:

var newElement = document.createElement('div');
newElement.textContent = 'Hello, World!';
document.body.appendChild(newElement);

Event Handling:

element.addEventListener('click', function() {
    alert('Element clicked!');
});

3. Java and Web Technologies

Java Servlet Example

Servlets are Java programs that run on a web server and generate dynamic web content.

import java.io.*;
import javax.servlet.*;
import javax.servlet.http.*;

public class HelloServlet extends HttpServlet {
    protected void doGet(HttpServletRequest request, HttpServletResponse response) 
        throws ServletException, IOException {
        response.setContentType("text/html");
        PrintWriter out = response.getWriter();
        out.println("<html><body>");
        out.println("<h1>Hello from Servlet!</h1>");
        out.println("<script>document.body.style.backgroundColor = 'lightblue';</script>");
        out.println("</body></html>");
    }
}

JavaServer Pages (JSP) Example

JSP is a technology that allows embedding Java code in HTML pages to create dynamic web content.

<%@ page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8" %>
<!DOCTYPE html>
<html>
<head>
    <title>JSP Example</title>
</head>
<body>
    <h1>Hello from JSP!</h1>
    <script>
        document.body.style.backgroundColor = 'lightgreen';
    </script>
</body>
</html>

Java with Front-End Frameworks

Spring Boot + Thymeleaf:

Thymeleaf is a server-side template engine used with Spring Boot to generate HTML content dynamically.

<html xmlns:th="http://www.thymeleaf.org">
<body>
    <h1 th:text="'Hello, ' + ${name}"></h1>
    <button onclick="alert('Button clicked!')">Click Me</button>
</body>
</html>
