1. Introduction to HTML

What is HTML?

Definition: HTML is the standard markup language used to create the structure of web pages. It consists of a series of elements, which you use to enclose different parts of the content to make it appear a certain way.

Structure: HTML documents are structured with a tree-like hierarchy of elements.

Basic Structure of an HTML Document

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <header>
            <h1>Welcome to My Website</h1>
            <nav>
                <ul>
                    <li><a href="#home">Home</a></li>
                    <li><a href="#about">About</a></li>
                    <li><a href="#contact">Contact</a></li>
                </ul>
            </nav>
        </header>
        <main>
            <section id="home">
                <h2>Home Section</h2>
                <p>This is the home section of the page.</p>
            </section>
            <section id="about">
                <h2>About Section</h2>
                <p>This is the about section of the page.</p>
            </section>
            <section id="contact">
                <h2>Contact Section</h2>
                <p>This is the contact section of the page.</p>
            </section>
        </main>
        <footer>
            <p>© 2024 My Website</p>
        </footer>
    </body>
</html>

2. Introduction to CSS

What is CSS?

Definition: CSS is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls the layout, colors, fonts, and overall look of web pages.

CSS Syntax

/* This is a CSS comment */
selector {
    property: value;
    property: value;
}

Basic CSS Properties

Text Formatting:

color: Sets the text color;
font-size: Sets the size of the font;
font-family: Specifies the typeface;

Box Model:

width: Sets the width of an element;
height: Sets the height of an element;
padding: Space between the content and the border;
border: Sets the border properties;
margin: Space outside the border;

Layout:

display: Specifies the display type (block, inline, flex, grid);
position: Specifies the positioning scheme (static, relative, absolute, fixed, sticky);
top, right, bottom, left: Specifies the position offsets;

Backgrounds and Borders:

background-color: Sets the background color;
background-image: Sets a background image;
border-radius: Rounds the corners of the border;

Flexbox Layout:

display: flex; Enables flexbox layout;
flex-direction: Defines the direction of the flex container's main axis (row, column);
justify-content: Aligns items along the main axis;
align-items: Aligns items along the cross axis;

Example CSS

/* Styling the body */
body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    margin: 0;
    padding: 0;
    background-color: #f4f4f4;
}

/* Styling the header */
header {
    background-color: #333;
    color: #fff;
    text-align: center;
    padding: 1em 0;
}

header nav ul {
    list-style: none;
    padding: 0;
}

header nav ul li {
    display: inline;
    margin: 0 10px;
}

header nav ul li a {
    color: #fff;
    text-decoration: none;
}

main {
    padding: 20px;
    max-width: 800px;
    margin: 0 auto;
}

section {
    margin-bottom: 20px;
    padding: 20px;
    background-color: #fff;
    border: 1px solid #ddd;
}

footer {
    text-align: center;
    padding: 10px 0;
    background-color: #333;
    color: #fff;
}

4. Advanced CSS Styling

Selectors

Universal Selector:

* {
    box-sizing: border-box;
}

Class Selector:

.className {
    color: blue;
}

ID Selector:

#idName {
    background-color: yellow;
}

Attribute Selector:

a[href*="example"] {
    color: red;
}

Pseudo-Classes and Pseudo-Elements

Pseudo-Classes:

a:hover {
    color: green;
}
input:focus {
    border: 2px solid blue;
}

Pseudo-Elements:

p::first-line {
    font-weight: bold;
}
p::after {
    content: " - End of paragraph";
}

CSS Flexbox and Grid

Flexbox Example:

.container {
    display: flex;
    justify-content: space-around;
    align-items: center;
}
.item {
    flex: 1;
    text-align: center;
}

Grid Example:

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}
.grid-item {
    padding: 20px;
    background-color: #ddd;
    text-align: center;
}

5. Responsive Design

Media Queries

Basic Syntax:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

Example:

.container {
    display: flex;
    flex-wrap: wrap;
}
.item {
    flex: 1 1 calc(33.333% - 20px);
    margin: 10px;
}

@media (max-width: 768px) {
    .item {
        flex: 1 1 calc(50% - 20px);
    }
}

@media (max-width: 480px) {
    .item {
        flex: 1 1 100%;
    }
}

Responsive Frameworks

Bootstrap:

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css">

<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/foundation/6.4.3/css/foundation.min.css">

6. Best Practices and Tips

Organize Your Code:

  • Use meaningful class and ID names.
  • Keep CSS in separate files for better maintainability.

Use Semantic HTML:

Use semantic tags to enhance accessibility and SEO.

Optimize Performance:

  • Minimize CSS and JS files.
  • Use sprites for images.

Test Responsiveness:

Use browser tools to test on different screen sizes.
